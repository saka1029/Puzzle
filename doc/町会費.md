# 町会費

## 問題
2026年度の町会費を収集した。
条件は以下の通り。

* 12戸から7800円ずつ徴収した。
* 支払いはすべて5000円札、1000円札、500円玉、100円玉の組み合わせのみで行われた。
* 8000円もらって200円お釣りを渡した人が何人かいた。
* 500円玉を2個以上払う人はいなかった。
* 徴収の結果、以下の紙幣および硬貨が集まった。

|硬貨・紙幣|枚数|金額|
|-|-:|-:|
|5000|9|45,000|
|1000|40|40,000|
|500|11|5,500|
|100|31|3,100|
|計||93,600|

## 支払いのパターン

支払いのパターンは以下の6通りであると考えられる。

|変数|5000|1000|500|100|
|----|---:|---:|--:|--:|
|a|1|2|1|3|
|b|1|2|0|8|
|c|1|3|0|-2|
|d|0|7|1|3|
|e|0|7|0|8|
|f|0|8|0|-2|

```
a = iota0 12
```


## 答え
|変数|5000|1000|500|100|解1|解2|
|----|---:|---:|--:|--:|---|---|
|a|1|2|1|3|8|9|
|b|1|2|0|8|0|0|
|c|1|3|0|-2|1|0|
|d|0|7|1|3|3|2|
|e|0|7|0|8|0|0|
|f|0|8|0|-2|0|1|

## 考察
2つの解はいずれも変数aおよびbがゼロであることから、
100円玉を8枚支払った人はいなかったことがわかる。
